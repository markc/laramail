About ai4me

A Laravel 12 + React 19 application featuring dual carousel sidebars, AI chat with multi-provider LLM support, and reusable admin datatables.

Tech Stack

  • Laravel 12 + Inertia 2
  • React 19 + TypeScript
  • Tailwind CSS 4
  • Prism PHP for LLM
  • SQLite + Vite 7
